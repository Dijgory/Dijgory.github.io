是ES6新增的内容，可以在函数调用/数组构造时，将数组表达式或者string在语法层面上展开

还可以在构造对象时，将对象表达式按照key-value的方式展开

也就是可以实现，将数组或者对象去掉[],{}的方式展开

数组的复制

let arr1 = [1,2,3]

let arr2 = […arr1]

这样不会指向一个数组，跟arr1.concat()一样的

数组的合并

let arr3 = […arr1,…arr2]

对象跟数组操作一样复制和合并

JSON.stringify() 方法用于将 JavaScript 值转换为 JSON 字符串。

var str = {“name”:”菜鸟教程”, “site”:”http://www.runoob.com"}

str_pretty2 = JSON.stringify(str, null, 4) //使用四个空格缩进
