CSS_position

代码
  • CSS
  • 学习

1:相对定位 position:relative

相对于元素在浏览器默认位置的定位，偏移都是相对于它本身的位置

2:绝对定位 position:absolute

相对于父元素（定义有position的父元素，position:relative , position:absolute,position:fiex这些都可以）定位，若父元素没有position那么就相对于浏览器窗口定位

3: z-index定义层序的，大的显示在上面，其中比较的两个属性都要有position属性

4：position：fixed 在固定的位置不变（与z-index连用）特别在窗口滑动的时候

5：position：sticky
