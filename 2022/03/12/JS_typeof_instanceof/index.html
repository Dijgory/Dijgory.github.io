2:使用instanceof(检测对象之间的关联性)

eg：

a = new Number(888)

a instanceof Number 为true

但左边的操作数要为引用类型值，或者为实例，右边一定要是函数

888 instanceof Number 为false

[1,2] instanceof Array 为true
